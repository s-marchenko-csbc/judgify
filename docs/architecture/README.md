# Judgify Architecture

This folder contains PlantUML diagrams for the Judgify platform.

Ця папка містить PlantUML діаграми архітектури та ключових сценаріїв Judgify.

## C4 Diagrams

- [System context](c4/c4-context.puml)
- [Container view](c4/c4-container.puml)
- [Backend component view](c4/c4-component-backend.puml)
- [Frontend component view](c4/c4-component-frontend.puml)

## Sequence Diagrams

- [Join competition](sequences/join-competition.puml)
- [Create and publish competition](sequences/create-and-publish-competition.puml)
- [Submit work](sequences/submit-work.puml)
- [Manual judging](sequences/manual-judging.puml)
- [Peer review](sequences/peer-review.puml)
- [Results publication](sequences/results-publication.puml)
- [Admin management](sequences/admin-management.puml)

The diagrams avoid external PlantUML includes so they can render in basic PUML tooling.

Діаграми не залежать від зовнішніх PlantUML include-файлів, тому мають відкриватися у базових PUML renderer-ах.
