# Журнал змін

Документ фіксує помітні продуктові та інженерні зміни Judgify.

## Поточна версія

- Додано повну продуктову специфікацію.
- Документацію перенесено в окрему папку `docs/`.
- Додано українську та англійську структуру документації.
- Додано PlantUML архітектурні та сценарні діаграми.
- Додано вимоги до майбутнього забезпечення якості ПЗ.

## Правила ведення

- Записувати зміни, які впливають на користувацькі сценарії, API, модель даних або deploy.
- Пов'язувати зміни зі специфікацією, якщо поведінка продукту змінюється.
- Не використовувати changelog як заміну acceptance criteria або тестам.
